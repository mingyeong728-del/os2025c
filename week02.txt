리눅스 명령어
-ls
현재 디렉터리의 파일과 폴더 목록을 출력함

-ls -a 
숨겨진 파일 폴더 보기

-touch파일명 
비어있는 파일 만들기

-cd폴더명 
폴더 바꾸기

-cd .. 
현재폴더에서 상위 폴더로 옮기기

-cat파일명
텍스트로 된 파일 문자 보여주기

-git 명령어
소스코드의 변경 이력을 관리하고, 협업을 위한 분산형 저장소를 다루는 데 사용

-git init
현재 디렉터리를 git저장소로 초기화

-git status
새로운 파일이 생긴지 기존의 파일이 수정된지 파일이 스테이지에 올라간지 보기(사건사고 보기)

-git add파일명
특정 파일 하나만 스테이징에 올린다

-git add.
모든 파일이 스테이지에 올린다

-git config --global user.email"개인이메일주소"
-git config --global user.name"필명"

-git cinfig --list
설정된 목록을 출력하기

-git commit -m "커밋에 남길 메시지"

-git log
커넷 몇 개 있나 보기

-git checkout 해시코드전체 또는 7자리
-git checkout -
커밋 상태로 갈 수도 있고
